# 🟣 拼豆图纸生成器

> 上传任意图片 → 像素化 → 一键生成可打印的拼豆图纸与颜色统计表

无论是动漫角色、游戏像素画、宠物照片还是自己画的图案，只需上传图片，设置网格大小，即可瞬间生成带颜色编号的拼豆图纸，附带每种颜色的珠子用量统计，直接对着拼就行。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📂 图片导入 | 支持 JPG / PNG / GIF / WebP 等常见格式 |
| 🎨 三套内置色板 | Perler 标准色、Hama 标准色、Artkal 方块色，另有全色彩模式 |
| 📐 灵活网格 | 15×15 到 60×60，适配各种尺寸的拼豆板 |
| 🔢 颜色编号 | 自动生成编号，每个格子里显示对应颜色编号，方便对照 |
| 📊 珠子用量统计 | 精确统计每种颜色需要多少颗珠子，购物清单一目了然 |
| 🧹 智能减色 | 自动将图片颜色映射到最近的色板颜色，高频颜色优先保留 |
| 📐 可选辅助线 | 网格线、颜色编号、行列坐标均可自由开关 |
| 💾 导出 PNG | 保存图纸和图例为高清 PNG 图片 |
| 🖨️ 打印支持 | 一键打印，图纸和颜色图例自动排版在同一页上 |
| 🌍 中英文切换 | 界面支持中文 / 英文一键切换 |
| 🚫 零依赖 | 单个 HTML 文件，无需安装，无需联网，双击浏览器打开即用 |

## 🚀 使用方法

### 第一步：上传图片

点击左侧 **「📂 上传图片」** 按钮，选择你想要制作的图片。可以是：

- 动漫角色
- 游戏角色或像素画
- 宠物照片
- 表情包
- 自己画的图案
- 任何你喜欢的图片

### 第二步：设置参数

| 参数 | 说明 | 建议 |
|------|------|------|
| **网格大小** | 决定图纸的精细程度 | 新手推荐 29×29（标准板尺寸） |
| **最大颜色数** | 限制使用的颜色种类 | 越少越容易制作，推荐 8~16 种 |
| **颜色板** | 选择你使用的珠子品牌 | 买了哪个牌子就选哪个 |

### 第三步：生成图纸

点击 **「✨ 生成图纸」**，等待几秒，右侧就会显示完整的拼豆图纸。

- 每个格子代表一颗珠子
- 格子内的数字是颜色编号
- 左侧颜色图例显示每种编号对应的颜色和用量

### 第四步：打印或保存

- 点 **「💾 保存图纸」** 导出为 PNG 图片
- 点 **「📋 保存图例」** 单独保存颜色统计表
- 点 **「🖨️ 打印」** 直接打印（图纸 + 图例自动排版）

### 第五步：开始拼豆

1. 把打印出来的图纸放在旁边
2. 对照图纸上的颜色编号和图例
3. 一行一行在拼豆板上摆放珠子
4. 摆完后用熨斗烫平定型

# 🟣 电子拼豆 — Perler Bead Studio

> 用 Python 制作的拼豆图纸编辑器，零依赖，开箱即用。
> 告别手画草稿，在电脑上设计图案，精确到每一颗珠子。

---

## 这是什么？

电子拼豆是一个桌面端的拼豆图纸设计工具。你可以像在画布上画画一样，用鼠标在网格上一格一格地摆放彩色珠子，设计出属于自己的拼豆图案。

设计完成后，导出颜色统计表，就知道每种颜色需要买多少颗珠子了。

---

## 为什么做这个？

| 痛点 | 电子拼豆怎么解决 |
|------|-----------------|
| 手画图纸太慢，画错了一擦一大片 | 撤销/重做，最多回退 50 步 |
| 颜色对不准，买回来的珠子色差大 | 内置 Perler / Hama / Artkal 三大品牌真实色板 |
| 数珠子数到眼花 | 一键导出颜色统计表，每种颜色需要几颗一目了然 |
| 纸质图纸弄丢了就白画了 | 保存为项目文件，随时打开继续编辑 |
| 想在网上找灵感但没有趁手的工具 | 免费、轻量、秒启动 |

工作流程
第 1 步：选择网格大小
        └─ 根据你手上的拼豆板选择对应尺寸

第 2 步：选择色板
        └─ 你买的哪个牌子的珠子就选哪个色板

第 3 步：设计图案

        └─ Ctrl+S 保存为 .pbd 文件

        └─ 文件 → 导出颜色统计

        └─ 对照电脑屏幕上的图纸
        └─ 一行一行在拼豆板上摆珠子
        └─ 摆完熨斗烫平定型
